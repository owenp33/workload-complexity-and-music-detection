# workload-complexity-and-music-detection

# Description
This repository contains notebooks of a Data Mining/Machine Learning project done to predict workload complexity and music type from physiological, hemoglobin, and behavioral signals. The data comes from Saman Khazaei, Srinidhi Parshi, Alam, S., Amin, M. R., & Faghih, R. T., found on Physionet.org (https://physionet.org/content/multimodal-nback-music/1.0.0/)
